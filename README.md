# 📊 Inteligência de Mercado

Análise e visualização de dados de mercado com **Python**, **Pandas** e **Jupyter Notebooks**.

## 🎯 Objetivo

Este projeto fornece ferramentas e análises para:
- 📈 Análise de tendências de mercado
- 💹 Visualização de dados econômicos
- 🔍 Insights baseados em dados
- 📉 Previsões e forecasting

## 🚀 Como Começar

### Pré-requisitos
```bash
Python 3.8+
Jupyter Notebook
pandas
plotly
numpy
```

### Instalação
```bash
# Clone o repositório
git clone https://github.com/samuelmarquesgit/InteligenciaDeMercado.git
cd InteligenciaDeMercado

# Instale as dependências
pip install -r requirements.txt

# Inicie o Jupyter Notebook
jupyter notebook
```

## 📁 Estrutura do Projeto

```
InteligenciaDeMercado/
```
