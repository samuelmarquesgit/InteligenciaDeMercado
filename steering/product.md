## Visão do Produto

**"Transformar dados brutos de vendas em insights acionáveis para o varejo, através de um pipeline Python reproduzível, modular e educativo."**

---

## Posicionamento

| Aspecto | Definição |
|---|---|
| **Para quem** | Analistas de Dados Júnior, estudantes de IA/Data Science |
| **O que é** | Pipeline de análise de dados de vendas 100% em Python |
| **Diferencial** | Educativo + profissional: código limpo, modular, documentado e versionado |
| **Contexto de uso** | Portfólio acadêmico e showcase de habilidades Python para dados |

---

## Objetivo Principal

Demonstrar domínio prático dos conceitos do Módulo 01 de IA para Análise Preditiva (SCTEC), produzindo um artefato de portfólio que prove:

1. Capacidade de construir pipelines de dados do zero em Python puro
2. Domínio aplicado de pandas, NumPy, Matplotlib e Seaborn
3. Aplicação de OOP (classes, herança, encapsulamento, method chaining)
4. Organização profissional com GitFlow, commits semânticos e documentação técnica
5. Clareza de comunicação: console output formatado, README profissional, vídeo técnico

---

## Restrições de Produto (v1.0)

| Restrição | Razão |
|---|---|
| Arquivo único (`salesinsight.py`) | Requisito explícito do mini-projeto |
| Dataset máximo de 200 linhas | Exercício didático — escala controlada |
| Sem ML real (apenas crescimento composto) | Escopo do Módulo 01 (Python base + pandas/NumPy) |
| Sem interface gráfica | CLI é suficiente para o objetivo avaliativo |
| Sem banco de dados | Dados em CSV/JSON conforme requisito |
