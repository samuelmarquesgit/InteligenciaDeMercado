# Steering — Produto

> Direcionamento estratégico do produto SalesInsight PY

---

## Visão do Produto

**"Transformar dados brutos de vendas em insights acionáveis para o varejo, através de um pipeline Python reproduzível e educativo."**

---

## Posicionamento

| Aspecto | Definição |
|---|---|
| **Para quem** | Analistas de Dados Júnior, estudantes de IA/Data Science |
| **O que é** | Pipeline de análise de dados de vendas em Python |
| **Diferencial** | Educativo + profissional: código limpo, modular, documentado e versionado |
| **Contexto de uso** | Portfólio acadêmico e showcase de habilidades em Python para dados |

---

## Objetivo Principal

Demonstrar domínio prático dos conceitos do Módulo 01 de IA para Análise Preditiva (SCTEC), produzindo um artefato de portfólio que prove:

1. Capacidade de construir pipelines de dados do zero
2. Domínio de pandas, NumPy, Matplotlib/Seaborn
3. Aplicação de OOP (classes, herança, encapsulamento)
4. Organização profissional com GitFlow, commits semânticos e documentação

---

## Restrições de Produto (v1.0)

| Restrição | Razão |
|---|---|
| Arquivo único (`salesinsight.py`) | Requisito do mini-projeto |
| Dataset máximo de 200 linhas | Exercício didático — não é Big Data |
| Sem ML real (apenas média móvel) | Escopo do Módulo 01 (Python base + pandas/NumPy) |
| Sem interface gráfica | CLI é suficiente para o objetivo |
| Sem banco de dados | Dados em CSV/JSON conforme requisito |

## Público e Tom de Comunicação

- **Código:** nomes de variáveis e funções em **português** (requisito da disciplina)
- **Docstrings:** concisas, explicando o "o quê" e não o "como"
- **Console output:** formatado, legível, com separadores visuais
- **README:** profissional mas acessível — para um avaliador que lê em 2 minutos
- **Vídeo:** informal mas técnico — demonstrar fluidez sem leitura de roteiro
