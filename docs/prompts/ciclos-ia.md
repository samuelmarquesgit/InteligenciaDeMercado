# Ciclos de IA — SalesInsight PY

> Rastreabilidade de todas as interações com IA no desenvolvimento do projeto

---

## O que é este arquivo?

Documenta cada ciclo de uso de IA (Claude, ChatGPT, Copilot, etc.) durante o desenvolvimento do SalesInsight PY, garantindo **transparência, rastreabilidade e autoria responsável** — fundamental para não caracterizar plágio, conforme as diretrizes do mini-projeto.

## Diretrizes de Uso Responsável da IA

1. **Nunca copiar código sem entender** — todo código gerado por IA deve ser lido, compreendido e adaptado
2. **Documentar cada ciclo** neste arquivo
3. **Ser capaz de explicar** qualquer linha de código no vídeo de demonstração
4. **A IA é um assistente** — o raciocínio, as decisões e o entendimento são do desenvolvedor
5. **Consultar materiais e documentação** é permitido e incentivado pela disciplina
