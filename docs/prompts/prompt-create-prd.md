# Prompt — Criação do PRD

> Rastreabilidade do prompt usado para gerar o PRD do projeto

---

## Prompt Utilizado

```
Leia o documento base do mini-projeto avaliativo (PDF da disciplina de IA para
Análise Preditiva - SCTEC, Módulo 1, Semana 08) e crie um PRD (Product Requirements
Document) profissional para o projeto SalesInsight PY.

O PRD deve conter:
- Visão do produto e sumário executivo
- Problema e solução
- Objetivos e métricas de sucesso com tabela
- Todos os 14 Requisitos Funcionais (RF01–RF14) documentados
- Requisitos não funcionais (portabilidade, reprodutibilidade, legibilidade)
- Fora de escopo (v1.0)
- Dependências e tabela de riscos
- Checklist de entregáveis obrigatórios
```

---

## Modelo de IA Utilizado
